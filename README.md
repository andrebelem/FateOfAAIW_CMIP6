## Overview

This repository contains the datasets, analysis scripts, and supplementary material used in the study assessing future changes in Antarctic Intermediate Water (AAIW) using CMIP6 simulations under SSP scenarios.

The workflows include:

- Processing CMIP6 ocean fields (temperature, salinity, density).
- Detection of the AAIW core using salinity minima.
- Construction of volumetric Tâ€“S diagrams.
- Heave and spice decomposition across latitudeâ€“longitude grids.
- Longitudinal ensemble statistics.
- Map visualizations and vertical sections.
  
*Note:* Large CMIP6 raw model files are not stored here. They can be downloaded from the Earth System Grid Federation (ESGF).
